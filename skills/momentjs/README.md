# Moment.js Skill

Covers Moment.js 2.x — date/time parsing, formatting, manipulation, querying, durations, timezones (moment-timezone), locales, and plugin integration.

## Contents

## Notes

Moment.js is in maintenance mode. For new projects, consider Luxon, Day.js, or date-fns. This skill applies fully to existing Moment.js code.
