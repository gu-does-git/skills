# Select2 v3 Skill

Covers Select2 v3.5 — searchable select/tag component for ServiceNow Service Portal widgets. Init with AngularJS, sync model changes, cascading dropdowns, modals, events, AJAX data loading, and i18n.

## Contents

- **SKILL.md** — full skill with AngularJS integration patterns, init, sync, cascading, modals, placeholder, i18n, pitfalls
- **references/**
  - `select2-3.5.md` — complete API reference: constructor options, methods, events, data format, AJAX config, CSS classes

## Key Topics

- AngularJS + Select2 sync (`change.select2` event)
- Template init via `ng-init` + `ng-change`
- Multi-select and single-select patterns
- Programmatic value changes and cascading dropdowns
- Select2 inside Bootstrap modals
- i18n / translations with locale files
- Placeholder + `allowClear` interaction
